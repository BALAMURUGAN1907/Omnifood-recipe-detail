# Recipe Card Page — Hyderabadi Biryani

A fully responsive recipe detail page built with pure HTML5 and CSS3,
showcasing a Hyderabadi Biryani recipe with ingredients, step-by-step
cooking instructions, and nutrition facts.

## Live Demo

https://omnifood-recipe-detail.netlify.app

## Features

- Sticky navigation bar with mobile hamburger menu
- Full-width hero section with overlay text on background image
- Ingredient cards with images, pricing, and "Buy" buttons
- Numbered step-by-step cooking instructions with hover animations
- Nutrition facts grid (Calories, Protein, Carbs, Fat)
- Fully responsive across desktop, tablet, and mobile
- Footer with contact info and quick links

## Built With

- HTML5 — semantic markup
- CSS3 — Flexbox, Grid, custom properties
- Responsive Design — media queries for multiple breakpoints
- Ionicons — for social and menu icons

## What I Learned

- Building a two-column responsive layout using CSS Grid
- Creating an overlay badge effect using `::before` and `::after`
- Implementing a mobile navigation menu without JavaScript
- Structuring real-world content cleanly
